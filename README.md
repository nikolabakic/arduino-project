# Arduino Project

Personal Arduino sketches, experiments, and components.

## Structure

```
arduino-project/
├── arduino-project.ino   # Main sketch
├── src/                  # Additional source files
└── lib/                  # Local libraries
```

## Requirements

- Arduino IDE or compatible toolchain
- Board: as specified per sketch

## Usage

Open `arduino-project.ino` in Arduino IDE, select your board and port, then upload.
